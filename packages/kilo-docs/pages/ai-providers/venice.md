# Using Venice AI With Kilo Code

Venice AI provides access to a broad catalog of text and multimodal models, including reasoning-capable variants, through a single API.

**Website:** [https://venice.ai/](https://venice.ai/)  
**Docs:** [https://docs.venice.ai/](https://docs.venice.ai/)

## Getting an API Key

1. **Sign Up/Sign In:** Go to [Venice AI](https://venice.ai/) and sign in to your account.
2. **Open API Settings:** Navigate to your account API key section.
3. **Create a Key:** Generate a new key for Kilo Code.
4. **Copy the Key:** Copy and store the key securely.

## Configuration in Kilo Code

1. **Open Kilo Code Settings:** Click the gear icon ({% codicon name="gear" /%}) in the Kilo Code panel.
2. **Select Provider:** Choose **Venice AI** from the provider list.
3. **Enter API Key:** Paste your Venice API key.
4. **Select Model:** Choose the Venice model you want to use.

## Tips and Notes

- **Reasoning models:** Venice offers reasoning-enabled models; pick model variants based on latency vs depth needs.
- **Model selection:** Venice includes multiple upstream model families (for example, Grok, Qwen, and others), so test a few models for your workflow.
- **Pricing:** Refer to Venice pricing/docs for the latest token pricing and limits.
